# MUNEZERO Alpha — Luxury Tech Portfolio

> A high-performance, dark-mode portfolio with a premium architectural feel. Built with **React**, featuring a "Dark Luxury-Tech" aesthetic, custom animations, and zero heavy dependencies.

## ✨ Features

| Feature | Description |
|---|---|
| 💎 **Dark Luxury Aesthetic** | Deep navy base with cyan glow and gold accents |
| 🪟 **Glassmorphism Design** | Frosted glass navigation with CSS backdrop filters |
| 🖱️ **Interactive Cursor** | Custom double-layered cursor that follows mouse movement |
| 🎞️ **Scroll Reveal Animations** | Sections fade and slide into view on scroll |
| ⌨️ **Typewriter Effect** | Dynamic header cycling through professional roles |
| 📱 **Fully Responsive** | Optimized for mobile, tablet, and wide desktop screens |
| ⚡ **Zero Heavy Dependencies** | Lightweight and fast — no bulky UI libraries |

---

## 🛠️ Tech Stack

- **React 18+** — Component-based architecture with hooks
- **CSS3** — Custom properties, animations, and glassmorphism effects
- **JavaScript (ES6+)** — Intersection Observer API, Mouse Event listeners
- **Fonts** — [Inter](https://fonts.google.com/specimen/Inter) & [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) via Google Fonts
- **Icons** — [Font Awesome 6](https://fontawesome.com/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v16 or higher
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/munezero-alpha.git

# 2. Navigate into the project folder
cd munezero-alpha

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

The app will be running at `http://localhost:5173` (or `http://localhost:3000` depending on your setup).

### Build for Production

```bash
npm run build
```

The optimized output will be in the `/dist` folder, ready for deployment.
