# MUNEZERO Alpha - Luxury Tech Portfolio

A high-performance, dark-mode portfolio for **MUNEZERO Alpha**. Built with pure HTML5, CSS3, and Vanilla JavaScript, this project features a "Dark Luxury-Tech" aesthetic with custom animations and a premium architectural feel.

## 💎 Features
- **Dark Luxury Aesthetic**: Deep navy base with cyan glow and gold accents.
- **Glassmorphism Design**: Frosted glass navigation with backdrop filters.
- **Interactive Cursor**: Custom double-layered SVG-style cursor following mouse movement.
- **Scroll Reveal animations**: Sections fade and slide into view as the user scrolls.
- **Typewriter Effect**: Dynamic header cycling through professional roles.
- **Fully Responsive**: Optimized for all screen sizes from mobile to wide desktops.
- **Zero Dependencies**: Pure code with no heavy frameworks for instant loading.

## 🛠️ Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, Grid, Flexbox, Keyframe animations.
- **JavaScript (ES6+)**: Intersection Observer API, Mouse Event listeners.
- **Fonts**: Inter and Space Grotesk via Google Fonts.
- **Icons**: Font Awesome 6.
